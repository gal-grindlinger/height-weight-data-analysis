# Height and Weight Data Analysis

This project performs data analysis using Python, NumPy, Pandas, and Matplotlib.

## Overview
- Processed numerical data using NumPy
- Cleaned and analyzed real-world dataset of height and weight
- Converted measurement units (inches to cm, pounds to kg)
- Identified and handled anomalies
- Visualized data using histograms

## Technologies
- Python
- NumPy
- Pandas
- Matplotlib

## How to Run

1. Make sure the following files are in the same folder:
   - data_q1.csv
   - height_and_weight.csv

2. Run the script:
   python height_weight_analysis.py

## Output
- Histograms of grouped means
- Height distribution (cm)
- Weight distribution (kg)
